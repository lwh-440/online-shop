# 在线购物网站

一个基于 Flask 的完整 B2C 在线购物系统，支持顾客、销售、管理员三种角色，集成数据分析、推荐系统、退款流程等高级功能。

## 功能特性

### 顾客端
- 用户注册/登录/注销（邮箱验证码注册、密码强度校验）
- 商品浏览、搜索（分类实时筛选）
- 商品详情页：双推荐板块（“购买了此商品的人也买过” + “猜你感兴趣”）
- 购物车管理
- 模拟付款、确认收货
- 退款申请（可选择仅退款/退货退款，上传凭据）
- 邮件通知（注册验证码、订单确认、发货通知）
- 浏览行为自动记录

### 销售端
- 商品管理（添加/编辑/删除自己负责的商品，弹窗添加）
- 订单管理（查看所有订单，发货操作）
- 分类管理（创建/删除分类，删除时商品自动移至“其他”）
- 浏览日志、操作日志查看
- 销售统计（ECharts 图表）

### 管理员端
- 所有销售端功能（无商品归属限制，可操作全部商品）
- 用户管理（创建销售/管理员账号，重置密码）
- 销售业绩查询（按商品归属统计）
- 退款审核（查看退款详情、凭据，同意或拒绝并恢复原状态）
- 销售异常实时监控
- 数据可视化大屏（ECharts）
- 推荐系统维护（手动更新相似度）

### 数据分析与推荐
- **协同过滤推荐**：基于订单共现计算商品相似度，“购买了此商品的人也买过”
- **兴趣推荐**：基于同类别、关键词相似度及浏览热度推荐“猜你感兴趣”
- 用户浏览行为采集（停留时长）
- 销售趋势图（日/周/月）、热门商品排行榜
- 销售异常检测（当日销售额超过过去7天平均值2倍时告警）
- 操作日志、登录日志记录

## 技术栈

- **后端**：Python + Flask
- **数据库**：MySQL 8.0
- **前端**：HTML/CSS/JS + Jinja2 模板 + Bootstrap 风格自定义样式 + ECharts
- **邮件**：QQ邮箱 SMTP
- **部署**：Gunicorn + Nginx（Ubuntu 24.04）

## 快速开始（本地开发）

### 环境要求
- Python 3.8+
- MySQL 5.7+（推荐8.0）
- 本地 SMTP 服务（QQ邮箱开启授权码）
