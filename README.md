在线购物网站

一个基于Flask的完整在线购物系统，包含用户注册登录、商品展示、购物车、订单管理、邮件通知等功能。

功能特性

顾客功能
- 用户注册、登录、注销
- 商品浏览和搜索
- 购物车管理
- 订单创建和查看
- 邮件确认通知

管理功能
- 商品管理（添加、编辑、删除）
- 订单管理
- 销售统计
- 图片上传管理

 技术栈
后端: Flask, MySQL
前端: HTML, CSS, JavaScript, Jinja2模板
邮件: QQ邮箱SMTP
部署: Ubuntu, Nginx, Gunicorn

环境要求
- Python 3.8+
- MySQL 5.7+
- Ubuntu 20.04+
